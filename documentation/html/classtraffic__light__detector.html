template<typename LocationNetType, typename StateNetType>
class traffic_light_detector< LocationNetType, StateNetType >

Template class for detection of traffic lights and theirs states.

Template Parameters
LocationNetTypeType of CNN, which is used to localize traffic lights in images.
StateNetTypeType of CNN, which is used to detect traffic light state.
